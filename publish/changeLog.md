## v0.0.2

这是 `lux-music-mobile` 分支的首个 UI 与发布链路整合版本，主要完成品牌替换、多语言文案补齐、设置页能力增强，以及发布更新链路对齐。

### 新增与优化

- 我的页（Me）与底部导航文案接入多语言（简中 / 繁中 / English）
- 播放栏空状态文案接入 i18n（未播放、引导提示）
- 设置页新增语言切换入口，可直接在应用内切换显示语言
- 设置页新增个人信息相关文案与交互细节优化（头像选择、昵称编辑）
- 搜索交互优化：搜索提交后同步全局搜索关键词状态

### 品牌与展示调整

- 应用名称统一为 `lux-music-mobile`
- 首页抽屉头部品牌文案替换为 `lux-music-mobile`
- Android 应用名与图标资源更新，安装后显示新品牌
- README 更新为 Lux Music 分支说明

### 发布与版本

- 版本升级到 `0.0.2`（`versionCode: 74`）
- 后续更新说明与发布文案改为本仓库维护
